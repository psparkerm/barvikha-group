# БАРВИХА — Памятка проекта

## Бренд
Барвиха Lounge (юр. — Barvikha group). Сеть кальянных премиум-класса, основана **2017 в Туле**.
Слоган: **«Новый уровень привычного удовольствия»**.

## География
- **Открыто:** Москва (17 точек), СПб (1), Махачкала (1), Тула (2). Итого **21 заведение в 4 городах**.
- **Скоро:** Нижний Новгород, Саратов, Пенза (Q2 2026) · Ереван (Q3) · Ташкент (Q4).

## Стек
- Чистый HTML5 / CSS3 / Vanilla JS — без фреймворков, без сборщиков.
- Шрифты: **Montserrat** (заголовки) + **Inter** (текст) — Google Fonts. *(Бренд-бук требует Baron Neue, но открытая версия с кривой кириллицей — отложено до получения оригинала от дизайнера бренда.)*
- Хостинг: статика (любой хостинг). Локально — `python3 -m http.server 8000`.

## Архитектура страниц
- Каждая страница — **самодостаточный монолит**: inline `<style>` + inline `<script>` + контент.
- Общие: `announcements.js` подключён на всех, остальное — внутри HTML.
- `data.js` существует, но **не подключён** (готовится для будущего рефакторинга монолитов).

## Система анонсов
Файл `announcements.js`. Массив `ANNOUNCEMENTS` в начале. Подключение:
```html
<script src="announcements.js" data-page="index"></script>
```
Чтобы добавить событие — допиши запись в массив. Поля: `id`, `pages` (на каких страницах), `image`, `title`, `description`, `ctaPrimary` (с `action: 'match-booking' | 'booking' | 'url' | 'tel' | 'close'`). Скрипт сам инжектит модалку и кнопку-индикатор в hero, помнит закрытие в `sessionStorage`.

## Бронирование
- На главной — модалка `#bkModal`, открывается `openBooking()`. Принимает `opts.match=true` + `opts.announcement` для контекстной брони на событие.
- На `location.html` есть кнопки **Telegram / WhatsApp / tel:** в hero — динамически берут номер из `LOC_PHONES` map по `?city=` (см. JS в самом файле). Заполнить телефоны — единственное место.
- **Restoplace** — не подключён. Когда придут ссылки по локациям — заменить кнопки брони на их виджет.
