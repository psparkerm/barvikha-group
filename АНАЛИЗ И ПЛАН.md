# 1. АНАЛИЗ БРЕНД-БУКА

Источник: `Брендбук Барвиха ИТОГ.pdf` (71 страница).

### Бренд
- **Название:** Барвиха Lounge (юр. — Barvikha group).
- **Год основания:** 2017, Тула.
- **Категория:** сеть кальянных премиум-класса.
- **Миссия:** «выведение привычного удовольствия на абсолютно новый уровень».
- **Слоган:** **«Новый уровень привычного удовольствия»**.
- **Ценности:** традиции + современность · забота о госте · команда.

### Логотип
- Дерево 🌳 + надпись «БАРВИХА» (широкая капитель) + «LOUNGE» вразрядку.
- Заложены 3 смысла: самореализация · финансовая стабильность · семейные ценности.
- Основная версия — **золотое дерево** (тёплое сусальное золото) на тёмном/деревянном фоне.
- Есть знак отдельно (только дерево) — для иконок/фавикона.

# 3. ВЫВОДЫ ИЗ АНАЛИЗА

### Общие паттерны — это must-have (есть почти у всех)
1. **Видео в hero** — живая атмосфера, не статика.
2. **Sticky-кнопка брони** — повторяется и закреплена на всех экранах.
3. **Карточка локации** = крупное фото + город + теги + 2 кнопки (Меню / Бронь).
4. **Фильтр / переключатель локаций** — по типу, городу, статусу.
5. **Заявление масштаба числом** в первом экране (Soho House: «48 Houses»).
6. **Статусы заведений** — «открыто», «скоро», «opening 2027».
7. **Единый виджет брони** на всю сеть, выбор заведения — в модалке.
8. **Много «воздуха»** + медленные тайминги анимаций = ощущение люкса.

### Уникальные фишки — что можно «украсть»
- **Zuma / Big Mamma:** фильтры и теги на карточках (Караоке / VIP / Веранда).
- **Coya:** модалка «выбери заведение» для брони с любой страницы.
- **STK:** мини-форма брони (4 поля) прямо в hero.
- **Paris Society / Big Mamma:** карусели «Избранное» и «Скоро открываются» + countdown.
- **Bugatti:** гигантский ALL-CAPS заголовок + cinema-black + видео.
- **Buddha Bar:** фирменный плейлист (Spotify-вставка) как часть бренда.
- **Kyma:** декоративные SVG-разделители секций.
- **Nikki Beach:** навигация «тип объекта × город».
